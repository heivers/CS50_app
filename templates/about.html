The major steps to create this app

Data Preparation

For this model the Oxford-IIIT Pet Dataset from Tensorflow was used. It contains about 7400 images.

The dataset was split into a training, validation and test set.(80/10/10). All images were resized and rescaled (values between 0 and 1). Data augmentation was used on the training set.

Model Training

Initial training for 10 epochs on the MobilenetV2 model. Initial validation accuracy was < 2%. After training approximately 81%.

Fine-tuned the model for another 10 epochs, by continuing training but also setting the last 54 layers to trainable. Accuracy improved to about 87 %.

Conversion to TFLite + App

Converted the model to a TFLite Model, so I can use the TFLite Interpreter in the app.

Created the Interpreter in the Flask Application and build the different html pages around it. Used CSS from W3.CSS to style the App.
